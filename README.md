# Markdown Cheatsheet

## Text format
| Format        | Symbol    | Sample Code          |
| ------------- | --------- | -------------------- |
| Bold          | *         | *italic*             |
| Italic        | **        | **bold**             |
| Strikethrough | ~~        | ~~struck through~~   |
| Code          | `         | `code`               |

## Inserts
| Format        | Symbol    |
| ------------- | --------- |
| Link          | `[]()`    |
| Image         | `![]()`   |
| Quote         | >         |
| Line          | ---       |
| Code Block    | ```       |
